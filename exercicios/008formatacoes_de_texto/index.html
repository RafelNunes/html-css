Principais formatações

Nesta frase temos um termo em negrito usando a tag <B> (não semântico).

Nesta frase tenho um termo em destaque usando a tag <STRONG> (semântico).

Italico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (não semântico).

Nesta frase, temos um termo em ênfase usando a tag <EM> (semântico).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro paragrafo temos outro texto marcado no final com a tag <MARK>.

Texto grande e pequeno

Estamos criando um texto grande com a tag <BIG>(obsoleto) e um texto pequeno nesse paragrafo com a tag <SMALL>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag <DEL>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <INS>.

Existe também o sublinhado com a tag <U> (não semântico).

Texto sobrescrito

Para inserir coisas do tipo x20+, usamos a tag <SUP>.

Texto subscrito

Para inserir coisas do tipo H2O, usamos a tag <SUB>.
